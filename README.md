# Job Portal Backend (Spring Boot)

A RESTful Job Portal backend application built using Spring Boot and JPA.

## Features
- Create Job
- Get All Jobs
- Get Job By ID
- Exception Handling
- H2 In-Memory Database

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- H2 Database
- Maven

## API Endpoints

### Create Job
POST `/jobs`

### Get All Jobs
GET `/jobs`

### Get Job By ID
GET `/jobs/{id}`

## How to Run
1. Clone the repository
2. Open in IntelliJ
3. Run `JobPortalApplication`
4. Test APIs using Postman
